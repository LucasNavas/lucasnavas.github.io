Imagem 5

Grafo construído no aplicativo Gephi utilizando a distribuição Radial Axis com o parâmetro centralidade de Autovetor, note que os maiores valores(Reino Unido e França) não possuem o maior grau, porém suas conexões são de maior importância.

Imagem 7

Grafo construído no gephi com os parâmetros: tamanho do nó por centralidade de grau e cor do nó por centralidade de proximidade (sendo cores quentes para valores altos e cores frias para os baixos).

Imagem 8

Grafo construído no gephi com os mesmos parâmetros da imagem 5 porém utilizando o plugin circular layout, ordenado pelo Id atribuído, mostrando que os vértices adicionados posteriormente e que possuem menor grau, também possuem uma menor centralidade closeness.

Imagem 9

Grafo construído no gephi com os parâmetros: centralidade de grau para o tamanho do nó e coeficiente de clusterização para a cor do nó (cores quentes para valores altos e cores frias para valores baixos).

Imagem 10

Grafo construído no gephi com os mesmos parâmetros da imagem 8, utilizando a função circular layout ordenada pelo grau do vértice. Percebe-se que os nós com alto grau possuem baixo coeficiente de clusterização, e os com grau muito baixo possuem coeficiente nulo.
